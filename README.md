# Installation
To install the program run
- Master/Controller:
  ```
  wget https://raw.githubusercontent.com/DeadlyBraincell/CC_Spawner_Controller/main/Master.Lua
  Master.lua
  ```
- Slave/Spawner Node:
  ```
  wget https://raw.githubusercontent.com/DeadlyBraincell/CC_Spawner_Controller/main/Slave.lua
  Slave.lua
  ```
